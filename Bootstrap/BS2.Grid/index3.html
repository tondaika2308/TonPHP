CSS Responsive Grid Class

  1. col-* áp dụng cho màn hình mobile (từ 0 đến 575px) vd: col-1, col-2, col-3, col-4, ..., col-12
  2. col-sm-* áp dụng cho tablet (từ 576-676px) vd: col-sm-1, col-sm-2, col-sm-3, ..., col-sm-12
  3. col-md-* áp dụng cho laptop (từ 768-991px) vd: col-md-1, col-md-2, col-md-3, ..., col-md-12
  4. col-lg-* áp dụng cho màn hình pc (từ 992-1199px) vd: col-ld-1, col-ld-2, col-ld-3, ..., col-ld-12
  5. col-xl-* áp dụng cho màn hình siêu lớn (từ 1200px trở lên) vd: col-xl-1, col-xl-2, col-xl-3, ..., col-xl-12
