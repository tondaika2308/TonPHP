Thuộc tính style là một thuộc tính chỉ ra cách hiển thị cho 1 thẻ trong html
Thuộc tính style là 1 thuộc tính chung cho tất cả các thẻ html
Thuộc tính style phân tách các thuộc tính bằng dấu ; ví dụ background: aqua; color: red
